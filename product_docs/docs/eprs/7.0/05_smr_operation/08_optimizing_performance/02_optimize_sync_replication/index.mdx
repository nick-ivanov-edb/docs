<div id="optimize_sync_replication" class="registered_link"></div>

You can set various configuration options for improving synchronization replication performance.

These options apply only to the publication server. You set them in the publication server configuration file.

For configuration options that apply to publication databases configured with the log-based method of synchronization replication, see [Specifying a custom URL for an Oracle JDBC connection](../../../10_appendix/04_miscellaneous_xdb_processing_topics/01_publications_and_subscriptions_server_conf_options/05_specify_custom_url_for_oracle_jdbc/#specify_custom_url_for_oracle_jdbc).
