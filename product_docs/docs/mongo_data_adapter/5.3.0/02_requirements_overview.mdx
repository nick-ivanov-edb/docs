## Supported Versions

The MongoDB Foreign Data Wrapper is certified with EDB Postgres Advanced Server 10 and above.

## Supported Platforms

The MongoDB Foreign Data Wrapper is supported on the following platforms:

**Linux x86-64**

 -   RHEL 8/OL 8
 -   Rocky Linux 8/AlmaLinux 8
 -   RHEL 7/OL 7
 -   CentOS 7
 -   SLES 15
 -   SLES 12
 -   Ubuntu 20.04/18.04 LTS
 -   Debian 10.x/9.x

**Linux on IBM Power (ppc64le)**

 -   RHEL 8
 -   RHEL 7
 -   SLES 15
 -   SLES 12
