title: "Version 4.0"
---


New features, enhancements, bug fixes, and other changes in Migration Portal 4.0 include the following:

| Type | Description |
| ---- |------------ |
| Bug Fix | Use of DDL extractor generated files is now re-enabled and issues that were encountered after the 3.5.0 release are fully addressed.|
| Feature| Added a repair handler to remove the CACHE keyword from CREATE TABLE statements to make them compatible with EDB Postgres Advanced Server.|
