---
title: "Version 4.0"
---


New features, enhancements, bug fixes, and other changes in Migration Portal 4.0 include the following:

| Type | Description |
| ---- |------------ |
| Bug Fix | Use of DDL extractor generated files is now re-enabled and issues that were encountered after the 3.5.0 release are fully addressed.|
| Feature| Added a repair handler to remove the CACHE keyword from CREATE TABLE statements. Note that EDB Postgres Advanced Server does not support caching recent data blocks retrieved for a table in buffer memory (ERH-2011: ORGANIZATION_INDEX_COMPRESS).|
